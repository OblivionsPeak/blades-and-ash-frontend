# Blades &amp; Ash Studio — Social Media Templates

A designed set of 8 editable, square (1080&times;1080) social-media post templates in the studio's brand:
obsidian backgrounds, sparing gold accents, high-contrast Cormorant serif headlines and clean Jost
labels. Each carries the crossed-shears logo mark and a consistent footer lockup.

These are **SVG** files. All the copy lives in real text elements, so you can change the words after
importing — no design software required to edit the wording.

## How to use them in Canva

1. In Canva, click **Create a design** (Custom size &rarr; 1080 &times; 1080 px).
2. Go to **Uploads** in the left sidebar &rarr; **Upload files**, and select the `.svg` file you want.
   - Canva imports SVG. Drag the uploaded graphic onto your blank 1080&times;1080 canvas and resize it to fill.
3. Double-click any text to edit it (price, dates, service name, promo code, etc.).
4. For the **Before &amp; After** template, drop your photos directly onto the two dark "BEFORE"/"AFTER"
   boxes (send the photo to back / position it under the labels, or delete the placeholder box first).

### Preview without Canva
Just double-click any `.svg` to open it in a web browser — it renders exactly as designed.

## Fonts
The templates use **Cormorant** (display serif) and **Jost** (sans labels), both free Google Fonts.
When opened in a browser the fonts load automatically. **Canva may substitute the fonts on import** —
if so, select **Cormorant** and **Jost** from Canva's font menu (both are available in Canva) to match
the original look.

## Editing notes
- Colors used: obsidian `#0E0E10`, surface `#16161A`, gold `#C8A24B`, soft gold `#D8BC7E`,
  bone text `#EDE7DB`, muted `#9A938A`.
- Replace `bladeandash.com` and all placeholder text (prices, dates, codes) with your real details.
- Keep gold as a sparing accent — it reads as luxe against the near-black.
